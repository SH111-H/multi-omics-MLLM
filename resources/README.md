# Resources

This folder collects the practical material around the paper list: datasets, benchmark references, evaluation reminders, and field-level challenges.

## Index

| Page | What it covers |
| --- | --- |
| [datasets-and-benchmarks.md](datasets-and-benchmarks.md) | Reusable datasets, benchmark axes, benchmark papers, and baseline tool references |
| [challenges.md](challenges.md) | Open problems, evaluation pitfalls, and a review checklist for reading new papers |

## How To Use This Folder

Use `datasets-and-benchmarks.md` before reproducing a paper, choosing a benchmark, or comparing models across modalities. Use `challenges.md` when reviewing claims about interpretability, robustness, privacy, missing modalities, or biological validity.
