# Contributing

Thanks for helping improve this living survey. New papers, corrected metadata, code links, dataset links, and clearer taxonomy are all welcome.

## Add a Paper

Use the most relevant page under `papers/` or `models/`. If a paper fits multiple buckets, cross-list it with a short note instead of forcing a single category.

Please include:

- Method or paper name.
- Model family or main idea.
- Task and modalities.
- Dataset or source.
- Paper link.
- Code or resource link when available.
- Year and one concise note.

## Preferred Links

- Primary publisher page or DOI.
- arXiv/bioRxiv/medRxiv when no peer-reviewed version is available.
- Official GitHub, Hugging Face, documentation, or dataset portal.

## Style

- Keep table rows concise.
- Use consistent names such as `TCGA`, `CPTAC`, `scRNA-seq`, and `scATAC-seq`.
- Avoid duplicate entries unless cross-listing is useful for discovery.
- Add recent papers when they extend the central-dogma or multi-omics framing.

## Useful Checks

- Confirm that every paper/resource has a working link.
- Confirm that the year matches the linked version.
- Add a short caveat when a method is a preprint, benchmark-only result, or has no public code.
