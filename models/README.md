## When A Method Belongs Here

Add a model here when its main contribution is reusable pretraining or broad transfer across multiple downstream biological tasks.

Examples:

- DNA, RNA, or protein language models.
- Central-dogma unified sequence models.
- Single-cell and spatial foundation models.
- Model-sharing hubs with reusable pretrained checkpoints.

If a method is mainly a task-specific predictor or fusion architecture, list it under `papers/` and cross-link here only when that improves discovery.
