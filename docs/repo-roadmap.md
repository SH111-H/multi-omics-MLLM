# Repository Roadmap

This scaffold is designed to evolve into a standard review-style GitHub repository.

## Phase 1: Structure

- Build a readable top-level README
- Split methods by task family
- Separate models from datasets and challenges
- Preserve the central dogma perspective

## Phase 2: Literature expansion

- Reconstruct the full bibliography from the source review
- Normalize paper metadata, venue names, and years
- Add paper links, code links, and benchmark tags
- Add post-2024 papers missing from the source review

## Phase 3: Review-grade presentation

- Add timeline figures
- Add side-by-side comparison tables
- Add benchmark leaderboards where reproducible
- Add a short tutorial path for new readers
